# Chat Application (Spring Boot)

## 📌 Description
This is a real-time chat application developed using Spring Boot.

## 🚀 Features
- User authentication
- Real-time messaging
- REST APIs

## 🛠 Tech Stack
- Java
- Spring Boot
- MySQL
- Maven

## ▶️ How to Run
1. Clone the repository
2. Open in Eclipse or STS
3. Run the main class
